# 🥗 NutriCalc

O NutriCalc é uma calculadora nutricional desenvolvida com HTML, CSS e JavaScript.

O projeto calcula indicadores importantes para avaliação nutricional e fornece uma estimativa da distribuição diária de macronutrientes de acordo com o objetivo do usuário.

## Funcionalidades

- ✅ Cálculo do IMC
- ✅ Classificação do IMC
- ✅ Taxa Metabólica Basal (TMB)
- ✅ Gasto Energético Diário (GETD)
- ✅ Ajuste calórico para:
  - Emagrecimento
  - Manutenção
  - Ganho de massa
- ✅ Cálculo de proteínas
- ✅ Cálculo de carboidratos
- ✅ Cálculo de gorduras
- ✅ Geração de relatório em PDF

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- html2pdf.js

## Como executar

1. Clone o repositório.

```bash
git clone https://github.com/xcardxx/NutriCalc.git
```

2. Abra o arquivo `index.html` no navegador.
